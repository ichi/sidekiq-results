# Sidekiq::Results

Results for sidekiq jobs.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sidekiq-results'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sidekiq-results

## Usage

Open `Results` tab on your sidekiq web UI.
